Chrome extension for capturing website/reference data, editing, adding tags and generating bibtex to copy-paste into reference managers (e.g. Zotero, Paperpile). 
Copy the repository to your files
Open Chrome Extensions in Developer Mode: chrome://extensions/
"Load Unpacked" and select the folder. It should unpack and work in your extensions. 
Pin to the Extensions toolbar and use.
