Made in Claude Pro </br>
Chrome extension for capturing website/reference data, editing, adding tags and generating bibtex to copy-paste into reference managers (e.g. Zotero, Paperpile).</br> 
Copy the repository to your files: copy https://github.com/SJTYLR/bibtex-grabber and paste to https://download-directory.github.io/ </br>
Open Chrome Extensions in Developer Mode: chrome://extensions/</br>
"Load Unpacked" and select the folder. It should unpack and work in your extensions. </br>
Pin to the Extensions toolbar and use. </br>
